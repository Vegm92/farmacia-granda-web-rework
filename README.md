# Farmacia Granda

A modern, high-performance e-commerce and informative web platform built for a local pharmacy to bridge the gap between traditional healthcare and digital convenience.

## Overview

Farmacia Granda is a full-stack web application designed to provide patients with a seamless way to browse products, access health information, and interact with the pharmacy's services. It prioritizes speed, accessibility, and a clean user interface.

## Objective

The objective was to create a digital storefront that reflects the professional and caring nature of a neighborhood pharmacy while providing modern features like search, product categorization, and responsive design.

## Why we built it

- **Digital Presence**: To enable the pharmacy to reach customers beyond their physical location.
- **User Experience**: To provide a faster and more intuitive experience than traditional pharmaceutical portals.
- **Modern Stack**: To leverage the power of Next.js for SEO and performance, ensuring that health information and products are easily discoverable.

## Technology Stack

- **Next.js**: Framework for server-side rendering and static site generation.
- **React**: Component-based UI library.
- **Tailwind CSS**: Utility-first CSS framework for responsive design.
- **Shadcn UI**: High-quality, accessible UI components.
- **TypeScript**: Ensuring code reliability and maintainability.

## Features

- **Product Catalog**: Easily searchable and categorized list of pharmaceutical products.
- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **SEO Optimized**: Built with performance and search visibility in mind.
- **Modern UI**: Clean and minimal aesthetic using Geist font and custom design tokens.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Run the development server: `npm run dev`.
4.  Open [http://localhost:3000](http://localhost:3000) to view the result.
